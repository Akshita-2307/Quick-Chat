# Quick Chat
It is a dynamic web-based chat application made using HTML,CSS and Javascript.
It allows user to create their profile,view others profile,send messages, delete messages in a clean and responsive way.

## Features
- User can make their profile with unique usernames.
- Sidebar showming all available users.
- Private one-to-one conversation.
- Delete message for me and everyone.
- View profile of ourself and others.
- Responsive design.
- Local storage based data.
